# Tropical Monster Girls 中文優化模組

## 📖 專案簡介 (Introduction)

本專案旨在改善 **Tropical Monster Girls** 的官方中文翻譯品質。

雖然官方提供了中文選項，但內容明顯使用了機械翻譯（Machine Translation），導致語意生硬，嚴重影響遊戲體驗。

因此，我使用 AI 進行重新翻譯，並進行了人工校對與潤飾，確保文本通順並符合遊戲語境。


## 🛠️ 安裝方法 (Installation)

1. 下載本專案的 Localization_zh-cn.json 檔案。
   * *(由於遊戲中未包含繁體中文字體，因此繁體中文會有部分字無法正常顯示，會在請求官方加入繁體中文，目前請先下載 Localization_zh-cn.json 簡體中文版)*
3. 從 Steam 進到遊戲目錄中 (Tropical Monster Girls\Game Files)，並將下載的檔案改名並覆蓋掉遊戲原本的 Localization.json 檔案。
   * *(別忘了覆蓋前請先備份原本的 Localization.json 檔案)*
4. 啟動遊戲即可生效。
